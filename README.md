# **📚 Repositório de Estudos de Programação Orientada a Objetos com Python**

Este repositório contém exemplos e estudos dos principais conceitos de **Programação Orientada a Objetos (POO)** utilizando a linguagem **Python**.  
 O objetivo é servir como material de estudo, consulta e prática dos conceitos fundamentais de POO.

---

# **📁 Estrutura do Repositório**

.  
├── classes\_abstratas  
├── classes\_objetos  
├── construtoes\_destrutores  
├── heranca\_poo  
├── metodos\_de\_classe\_e\_metodos\_estaticos  
├── poo\_relacionamentos\_python  
└── python\_properties

Cada diretório representa um conceito da Programação Orientada a Objetos.
